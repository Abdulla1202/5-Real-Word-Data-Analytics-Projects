# 📱 Exploratory Data Analysis using SQL on Electronics Dataset

## 📌 Project Overview
Performed SQL-based data analysis on an electronics sales dataset to extract business insights.

## 🎯 Objectives
- Perform aggregations using SQL
- Identify top-selling products
- Analyze revenue trends
- Create business insights using queries

## 🛠 Tech Stack
- SQL (SSMS)
- Excel (for visualization)

## 📊 Key Insights
- Revenue contribution by category
- Monthly sales trend
- Top performing products

## 🚀 How to Run
1. Import dataset into SQL Server
2. Run provided SQL queries
3. Use Excel for charts if needed
